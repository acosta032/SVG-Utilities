SVG Utilities (Scalable Vector Graphics)

SVG es un ESTANDAR web para para almacenar información de graficos vectoriales basando en XML. SVG usa etiquetas para guardar información de

  • Formas
  • coordenadas
  • animaciones

Un SVG se encierra dentro de las etiquetas <svg> </svg>

Los atributos iniciales más comunes son:

  • version: Versión del estandar SVG de la W3C

  • viewBox: [x-min | y-min | x | y ]

    Es la posición y la dimensión del lienzo relativo de las dimensiones Widht y Height

    Si no se establece el lienzo es infinito

    x-min | y-min : Coordenadas Top - Left

    x | y : Son las dimensiones entre (comunes 0 0 100 100(

    Este atributo se puede usar en otras etiquetas como:

    • <marker>
    • <pattern>
    • <symbol>
    • <view>
    • <svg>

                svg version="1.1"
                width="100%"
                viewbox="0 0 600 300"
                xmlns="http://www.w3.org/2000/svg">
